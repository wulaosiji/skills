# 🛡️ 前置逻辑校验系统 v1.0

> **目标**：在给出答案前，主动发现逻辑漏洞，而非事后补救  
> **触发时机**：所有推荐、报价、数据分析类回复  
> **创建原因**：咖啡豆价格逻辑错误（预算300元≠能买真蓝山1kg）

---

## 📋 通用校验清单（发送前必查）

### 1. 价格合理性校验

**检查点**：
```
□ 商品价格是否符合市场常识？
□ 是否混淆了"正品"和"仿品/替代品"？
□ 是否考虑了品牌溢价 vs 普通品牌？
□ 是否有1kg vs 100g的单位陷阱？
```

**常见陷阱**：
| 场景 | 错误示例 | 正确认知 |
|------|---------|---------|
| **蓝山咖啡** | 1kg 200元 | 真蓝山1kg 500-1500元 |
| **iPhone** | 全新iPhone 15 3000元 | 官方价5999元起 |
| **茅台** | 飞天茅台500元 | 市场价2500-3000元 |
| **大闸蟹** | 阳澄湖1两蟹10元 | 正品2两蟹80-150元 |

**校验方法**：
- 快速搜索"XX商品 正常价格"
- 检查价格是否低于成本价（不可能）
- 确认单位（1kg vs 100g）

---

### 2. 真伪鉴别校验

**检查点**：
```
□ 是否区分了"真品"和"风味/拼配/替代品"？
□ 是否标明了原产地/认证标识？
□ 是否提醒了常见的假冒话术？
```

**常见话术陷阱**：
| 话术 | 实际含义 | 风险等级 |
|------|---------|---------|
| "XX风味" | 不是真XX，是调味 | 🟡 中等 |
| "XX拼配" | 可能含少量XX | 🟡 中等 |
| "XX同款" | 仿品 | 🔴 高 |
| "XX平替" | 替代品 | 🟢 低（如明说） |
| "XX品质" | 不是真XX | 🔴 高 |

**校验方法**：
- 看到"风味"、"拼配"、"同款"等词立即警觉
- 明确标注"真品"vs"替代品"

---

### 3. 需求-方案匹配校验

**检查点**：
```
□ 用户需求是否明确？（真蓝山 vs 蓝山风味）
□ 预算是否匹配需求？（300元 vs 真蓝山1kg）
□ 方案是否在预算内？
□ 是否给出了多个选项（A/B/C）？
```

**本次咖啡豆错误复盘**：
```
用户需求：蓝山咖啡豆 1kg 300元内
我的错误：推荐了"蓝山风味"但未明确说明不是真蓝山
正确做法：
  方案A：真蓝山小包装（300元/200g，明确说明量）
  方案B：蓝山风味1kg（150-280元，明确说明是拼配）
  方案C：牙买加高山咖啡1kg（真产地但非蓝山区域）
```

---

### 4. 数据一致性校验

**检查点**：
```
□ 不同来源数据是否一致？
□ 是否存在明显矛盾？
□ 时间戳是否合理（不是过期数据）？
```

**校验方法**：
- 交叉验证2-3个来源
- 发现矛盾时明确指出

---

### 5. 单位换算校验

**检查点**：
```
□ 是否统一了单位？（kg vs g，元 vs 美元）
□ 是否计算正确？（单价=总价/数量）
```

**常见单位陷阱**：
| 陷阱 | 示例 |
|------|------|
| **重量单位** | 1kg报价 vs 100g实际 |
| **货币单位** | 美元标价 vs 人民币理解 |
| **时间单位** | 日薪 vs 时薪（24h vs 8h） |

## ⚡ 快速自检流程

```
用户提问 → 理解需求 → 生成答案
              ↓
        【前置校验】
              ↓
    1. 价格是否合理？（常识检查）
    2. 是否混淆真伪？（关键词检查）
    3. 需求预算是否匹配？（逻辑检查）
    4. 单位是否一致？（换算检查）
              ↓
    发现问题 → 修正答案 / 给出多选项
    无问题 → 发送答案
              ↓
        【附加说明】
    - 明确标注"真品"vs"替代品"
    - 给出预算内多个选项
    - 提示潜在风险
```

## 📝 本次咖啡豆错误记录

**错误类型**：需求-方案不匹配 + 真伪混淆

**用户真实需求**：
- 商品：蓝山咖啡豆
- 规格：1kg
- 预算：300元内
- 隐含：大概率要"真蓝山"

**我的错误**：
1. 未先校验"真蓝山1kg"的市场价（实际500+，超预算）
2. 推荐了"蓝山风味"但未明确说明是替代品
3. 未给出A/B/C选项让用户选择

**正确做法**：
```
先校验：真蓝山1kg价格远超300元预算
再给出选项：
  A. 真蓝山小包装（300元/200g）
  B. 蓝山风味1kg（150-280元，拼配豆）
  C. 牙买加高山咖啡1kg（200-300元，真产地）
```

def generate_recommendation(product, budget):
    # 生成推荐前
    if product == "蓝山咖啡" and budget < 500:
        warning = "⚠️ 真蓝山1kg超预算，建议："
        options = [

            "B. 蓝山风味1kg（替代品）",
            "C. 牙买加高山咖啡1kg"
        ]
        return warning + "\n" + "\n".join(options)
