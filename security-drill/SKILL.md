# Security Drill Skill

攻防模拟测试技能 - 定期验证安全响应能力

## 概述

本技能用于定期进行安全攻防演练，测试和验证安全响应能力。

## 使用方式

1. 定义演练场景
2. 运行测试验证
3. 分析响应结果

## 演练类型

- 场景描述
- 攻击方式
- 预期响应
- 参考案例
- 严重程度评估
